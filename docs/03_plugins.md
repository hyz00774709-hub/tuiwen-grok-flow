# 插件与外部工具建议

## 已确定需要

- GitHub：保存流程、模板、配置、脚本、skill。
- Google Drive：保存参考视频、小说文本、音频、SRT、图片、视频片段、成片。

## 暂时不必增加

- Gmail / Outlook Email：除非后续要自动发片、发报告、发合作邮件。
- Google Calendar / Outlook Calendar：除非要排生产计划和提醒。
- Slack / Teams：除非工作室多人协作，需要自动通知审核。
- Notion：如果你已经习惯用 Notion 做项目看板，可以接；否则先别加，避免资料分散。
- Figma：如果后面要做一套很精细的 UI 面板视觉系统，再接也不迟。
- Box / SharePoint：你已经用 Google Drive 的话，暂时没必要。
- Atlassian Rovo：除非后面做团队级研发管理。

## 后续可能需要

第一阶段先不用额外插件。等流程稳定后，再考虑：

- Notion：做书单库、爆点库、项目状态看板。
- Slack / Teams：多人审核通知。
- Figma：系统面板、排行榜、弹窗 UI 设计资产。
