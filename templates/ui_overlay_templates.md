# 系统 UI 叠加模板

所有中文 UI 信息优先由代码模板叠加，不交给图片或视频模型生成。

## 常用模板

### 系统弹窗

字段：

- title
- status
- description
- options

### 属性面板

字段：

- name
- level
- attack
- defense
- speed
- skill
- rank

### 排行榜

字段：

- title
- rows
- highlight_row

### 直播间弹幕

字段：

- room_title
- viewer_count
- comments
- gift_effect

## 视觉规则

- 文字清晰，不能过细。
- 重要数字可用红色或金色。
- 面板透明度 70%-90%。
- 字幕与 UI 不要互相遮挡。
- 主字幕固定底部安全区。
